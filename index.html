Base Cycles

Process

Pick a fixed number N. For all 1 < n < N, convert N to base n and apply the following function \[ f(n) = n_b \] where \[ f(n) = n_b : b = \sum_{i=1}^{\lfloor log_{10}(n) + 1 \rfloor} d_i \] For each base b, create a node in the graph. If \[ f(n_a) = f(n_b) \] Then draw an edge from a to b.

Observations

\[ b = n: f(n_n) = 1\]

\[ b > n: f(n_b) = n \]
